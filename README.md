# Real-Time ChatBox Messenger (Flask + MySQL)

A real-time chat application built while exploring DBMS concepts.

## Features
- User registration & login
- Unique usernames
- Secure password hashing (SHA-256)
- User profiles
- Real-time messaging (AJAX polling)
- Unread message notifications
- Delete chat & delete account
- MySQL database integration

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS, JavaScript
